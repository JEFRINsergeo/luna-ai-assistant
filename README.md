# Luna AI Assistant

Luna is an Hybrid AI assistant with a built-in cybersecurity engine.

## Features

- AI chat assistant(Online+Offline)
- Process behavior monitoring
- Network monitoring
- Ransomware detection
- Download threat detection
- Security event logging

## Technologies Used

- Python
- Streamlit
- Ollama
- psutil
- threading
- Gemini

## Architecture

Luna uses a modular security engine with:

- Behavior Monitor
- Network Monitor
- Ransomware Detector
- Download Monitor
